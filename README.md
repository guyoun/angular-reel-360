# angular-reel

Simple Augular directive for [Reel](http://reel.org)

# Installation
`$ bower install angular-reel-360`

# Usage
```js
image = 'DSCN691.JPG'
images = "/DSCN####.JPG|691..702"
reel_control = {}
```

```html
<div angular-reel id="" images="images" image="image" use-magnify='true' control="reel_control"></div>
```
